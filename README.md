# Prodesk IT - Cash-Flow Dashboard 💰

This is my Sprint 02 project assigned during my internship at Prodesk IT. I built a functional Cash-Flow dashboard using Vanilla JavaScript where users can input their salary, log expenses, and view real-time balance updates.

## Live Site

🔗 https://harshmaurya-dev.github.io/prodesk-it-cashflow-dashboard/

## GitHub Repo

🔗 https://github.com/harshmaurya-dev/prodesk-it-cashflow-dashboard

---

## What I made

- Salary input form to set total salary
- Expense form to add expense name and amount
- Real-time Remaining Balance calculation (Salary - Total Expenses)
- Delete button on each expense to remove it instantly
- Data persistence using LocalStorage (salary and expenses stay saved after reload)
- Dynamic Pie Chart (Chart.js) showing Remaining Balance vs Total Expenses
- Input validation - blocks empty/negative values and shows error message
- Low balance warning - balance turns red if it drops below 10% of salary

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Chart.js
- LocalStorage API

No frameworks used. Only Vanilla JS as required in sprint.
